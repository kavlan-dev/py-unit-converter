# Unit Converter

## Описание проекта

Unit Converter - это веб-приложение для конвертации единиц измерения, поддерживающее три основных типа конвертации: длина, вес и температура. Приложение состоит из серверной части (backend) на Python с использованием FastAPI и клиентской части (frontend) на HTML/CSS/JavaScript.

## Решаемая проблема

Приложение решает проблему быстрой и удобной конвертации между различными единицами измерения, что особенно полезно для:
- Учебных целей
- Инженерных расчетов
- Путешествий (конвертация температур)
- Повседневных задач (приготовление пищи, строительные работы)

## Преимущества

- Простой и интуитивно понятный интерфейс
- Поддержка метрической и имперской систем измерения
- Быстрая обработка запросов благодаря современному стеку технологий
- Кроссплатформенность (работает в любом современном браузере)
- Легкость развертывания с помощью Docker

## Установка и использование

### Предварительные требования

- Docker и Docker Compose (для запуска через контейнеры)
- Python 3.13 (для локального запуска backend)

### Запуск с помощью Docker (рекомендуемый способ)

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd py-unit-converter
   ```

2. Запустите приложение:
   ```bash
   docker-compose up --build
   ```

3. Откройте браузер и перейдите по адресу: `http://localhost:8000`

### Локальный запуск (без Docker)

#### Backend

1. Перейдите в директорию backend:
   ```bash
   cd backend
   ```

2. Установите зависимости:
   ```bash
   poetry install
   ```

3. Запустите сервер:
   ```bash
   poetry run python src/py_unit_converter/main.py
   ```

#### Frontend

1. Откройте файл `frontend/index.html` в браузере
2. Или используйте локальный веб-сервер:
   ```bash
   cd frontend
   python -m http.server 8000
   ```

## Использование приложения

1. Выберите тип конвертации (Длина, Вес или Температура)
2. Введите значение для конвертации
3. Выберите исходную и целевую единицы измерения
4. Нажмите кнопку "Конвертировать"
5. Результат появится в соответствующем поле

## Примеры использования

### Конвертация длины
- 10 метров в километры = 0.01 км
- 12 дюймов в футы = 1 фут

### Конвертация веса
- 1000 грамм в килограммы = 1 кг
- 16 унций в фунты = 1 фунт

### Конвертация температуры
- 0°C в Фаренгейты = 32°F
- 100°C в Кельвины = 373.15 K

## API Документация

Backend предоставляет следующие эндпоинты:

### Конвертация длины
```
POST /api/convert/length
```

**Параметры запроса (JSON):**
```json
{
  "value": 10.0,
  "from_unit": "meter",
  "to_unit": "kilometer"
}
```

**Поддерживаемые единицы:**
- Метрические: millimeter, centimeter, meter, kilometer
- Имперские: inch, foot, yard, mile

### Конвертация веса
```
POST /api/convert/weight
```

**Параметры запроса (JSON):**
```json
{
  "value": 1000.0,
  "from_unit": "gram",
  "to_unit": "kilogram"
}
```

**Поддерживаемые единицы:**
- Метрические: gram, kilogram
- Имперские: pound, ounce

### Конвертация температуры
```
POST /api/convert/temperature
```

**Параметры запроса (JSON):**
```json
{
  "value": 0.0,
  "from_unit": "celsius",
  "to_unit": "fahrenheit"
}
```

**Поддерживаемые единицы:** celsius, fahrenheit, kelvin

## Технологии

### Backend
- **Язык**: Python 3.13
- **Фреймворк**: FastAPI
- **Зависимости**:
  - fastapi (>=0.129.0,<0.130.0)
  - uvicorn (>=0.40.0,<0.41.0)

### Frontend
- **HTML5** для структуры страницы
- **CSS3** для стилизации
- **JavaScript** для интерактивности
- **Fetch API** для асинхронных запросов

### Инфраструктура
- **Docker** для контейнеризации
- **Nginx** как обратный прокси
- **Docker Compose** для оркестрации

## Архитектура

Приложение использует клиент-серверную архитектуру:

1. Пользователь взаимодействует с frontend (HTML/CSS/JS)
2. Frontend отправляет AJAX-запросы к backend API
3. Backend обрабатывает запросы, выполняет конвертацию
4. Результаты возвращаются в формате JSON
5. Frontend отображает результаты пользователю
